# springboot-aws
'스프링 부트와 AWS로 혼자 구현하는 웹 서비스' 책 학습용 저장소

## 목표
- AWS 서비스를 활용하여 24시간 클라우드 컴퓨팅 서비스를 사용한다.
- CI/CD와 Nginx를 활용한 무중단 배포 서비스를 구축한다.

## 기술 스텍
- Spring boot
- Spring Security
- Oauth2
- AWS EC2, RDS, S3, CodeDeploy
- Travis CI
- Nginx
- Thymeleaf
